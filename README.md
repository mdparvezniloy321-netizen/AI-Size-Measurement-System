# 🎯 AI Real-Time Size Measurement System
This is a computer vision application built with **Python**, **YOLOv8**, and **Streamlit**. It detects objects in a video and estimates their real-world dimensions (Width and Height in centimeters) based on pixel-to-metric conversion.

## 🚀 Features
- Real-time Object Detection using YOLOv8.
- Pixel-to-centimeter coordinate conversion geometry.
- Interactive Web Interface via Streamlit for easy video uploads.

## 🛠️ Tech Stack
- **Language:** Python
- **AI Model:** YOLOv8 (Ultralytics)
- **Computer Vision:** OpenCV
- **Web Framework:** Streamlit
